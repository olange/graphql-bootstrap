# Poliscope DM

Poliscope Document Model: a quick proof-of-concept with GraphQL.

To launch the server:

```bash
$ npm start -s
```

Then open a browser at http://localhost:4000 and try following GraphQL query:

```json
{
  greeting1: hello
  greeting2: hello( who: "David")
}
```
